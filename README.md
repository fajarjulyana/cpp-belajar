## Prasyarat & Instalasi Compiler

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install build-essential g++ cmake make git
g++ --version
```

### macOS
```bash
xcode-select --install
brew install cmake
g++ --version   # sebenarnya clang di macOS
```

### Windows
- **MinGW-w64**: Download dari https://www.mingw-w64.org/  
- **MSVC**: Install Visual Studio 2022 (Community gratis)  
- **CMake**: https://cmake.org/download/

---

## Cara Kompilasi Cepat

```bash
# Kompilasi satu file
g++ -std=c++17 -Wall -o program main.cpp

# Dengan optimasi
g++ -std=c++17 -O2 -o program main.cpp

# Debug (dengan info debug)
g++ -std=c++17 -g -o program main.cpp

# Program yang butuh threading atau networking
g++ -std=c++17 -pthread -Wall -o program main.cpp
```
