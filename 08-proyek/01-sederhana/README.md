# Proyek Sederhana — C++

Kumpulan proyek kecil yang bisa diselesaikan dalam 1-3 hari untuk mengasah kemampuan C++ secara praktis.

---

## Daftar Proyek

| No | Proyek | File | Konsep |
|----|--------|------|--------|
| 01 | Kalkulator CLI | `kalkulator.cpp` | Fungsi, switch, loop |
| 02 | To-Do List | `todo.cpp` | Vector, struct, file I/O |
| 03 | Konverter Suhu | `konverter.cpp` | Fungsi, map, input |
| 04 | Manajemen Kontak | `kontak.cpp` | OOP, struct, sorting |
| 05 | Quiz Sederhana | `quiz.cpp` | File I/O, random, vektor |

---

## Proyek 01: Kalkulator Ilmiah CLI

Kalkulator baris perintah yang mendukung operasi dasar dan fungsi matematika.

**Fitur:**
- Operasi: +, -, *, /, %, ^
- Fungsi: sin, cos, tan, sqrt, log, abs
- Riwayat kalkulasi
- Mode loop interaktif

Lihat: [kalkulator.cpp](kalkulator.cpp)

---

## Proyek 02: To-Do List dengan File

Aplikasi manajemen tugas yang menyimpan data ke file teks.

**Fitur:**
- Tambah/hapus/tandai selesai tugas
- Prioritas (tinggi/sedang/rendah)
- Simpan ke file & muat saat startup
- Filter berdasarkan status

Lihat: [todo.cpp](todo.cpp)

---

## Proyek 03: Konverter Unit

Konversi berbagai satuan: suhu, panjang, berat, mata uang.

**Fitur:**
- Suhu: Celsius ↔ Fahrenheit ↔ Kelvin
- Panjang: meter, cm, inch, feet, mile
- Berat: kg, gram, pound, ounce
- UI menu interaktif

Lihat: [konverter.cpp](konverter.cpp)

---

## Proyek 04: Manajemen Kontak

Buku alamat sederhana dengan operasi CRUD.

**Fitur:**
- Tambah/edit/hapus kontak
- Cari berdasarkan nama/nomor
- Urutkan berdasarkan nama
- Export ke CSV

Lihat: [kontak.cpp](kontak.cpp)

---

## Proyek 05: Game Quiz

Kuis pilihan ganda dengan timer dan skor.

**Fitur:**
- Baca soal dari file JSON/teks
- Acak urutan soal
- Timer per soal
- Papan skor (leaderboard)

Lihat: [quiz.cpp](quiz.cpp)
